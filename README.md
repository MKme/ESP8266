### Wiring SD Card to WEMOS D1 Mini Board
| Pin On SD Reader | Pin on D1 Mini |
| ---------- |----------------|
| LED  | 3.3 or 5V depending on your display  |
| SCK  | D13 |
| SDA  | D11 (MOSI) |
| A0   | D8 |
| RST  | D9  |
| CS   | D10 |
|GND   | GND |
|VCC  | 3.3 or 5V depending on your display  |
