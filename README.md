### SPI SD Card to WEMOS D1 Mini
| Pin On SD Reader | Pin on D1 Mini |
| ---------- |----------------|
| SCK  | D5 |
| MISO| D6 |
| MOSI  | D7 |
| CS  | D8  |
|GND   | GND |
|VCC  | 3.3V ONLY!  |

### 0.96" OLED to WEMOS D1 Mini 
| Pin On OLED | Pin on D1 Mini |
| ---------- |----------------|
| SDA  | D2 |
| SCL| D1 |
|GND   | GND |
|VCC  | 3.3V ONLY!  |
